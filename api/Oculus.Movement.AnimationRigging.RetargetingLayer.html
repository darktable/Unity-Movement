    _enableTrackingByProxy

    Create proxy transforms that track the skeletal bones. If the skeletal bone transforms change, that won't necessitate creating new proxy transforms in most cases. This means any Animation jobs that track the skeletal bone transform can use proxies instead, which get re-allocated less often. Re-allocation would mean having to create new animation jobs.
